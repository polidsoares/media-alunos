Um simples programa em Python para calcular médias de alunos.

## Como Usar
Apenas execute pelo terminal:

```
python medias.py --entrada notas.txt --saida medias.txt
```

Onde a flag `--entrada` informa o arquivo de entrada e a flag `--saida` informa o arquivo de saída.
O arquivo de notas deverá estar no seguinte formato:

```
Aluno Um	8	8	9
Aluno Dois	9	7	6
```
